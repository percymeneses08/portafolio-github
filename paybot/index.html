PayBot

Es un ayudante de transacciones desarrollado para facilitar los procesos de pagos entre personas y comercios. Funciona con un Bot de Whatsapp para gestión de fondos, consulta de datos. También escribe y actualiza información de usuarios en una base de datos.

Los usuarios deben realizar el depósito en una cuenta para poder contar con fondos dentro del ecosistema de PayBot.
